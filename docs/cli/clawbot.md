# `alvasta-pro clawbot`

Legacy alias namespace kept for backwards compatibility.

Current supported alias:

- `alvasta-pro clawbot qr` (same behavior as [`alvasta-pro qr`](/cli/qr))

## Migration

Prefer modern top-level commands directly:

- `alvasta-pro clawbot qr` -> `alvasta-pro qr`
